The Grid


The grid can be created by using the row and column classes.

The xxs, xs, s, m, l, xxl, fill classes can be used if you want to customize the column size.

To not be used like Bootstrap (there are no col-x, col-xs, col-etc classes) or other CSS frameworks. This grid has been designed in order to handle just the common use cases.
